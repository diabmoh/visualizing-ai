Sources and simplifications

Northstar Observatory, its documents, people, dates, embeddings, similarity scores, and answers are fictional. Coordinates and rankings are illustrative, not outputs from a live embedding model.

This exhibit shows a common dense-retrieval RAG pipeline. Real systems may use lexical or hybrid search, metadata filters, reranking, query rewriting, iterative retrieval, different chunking strategies, and access controls.

  • Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks for the foundational RAG architecture
  • Dense Passage Retrieval for Open-Domain Question Answering for dense retrieval
  • BEIR for heterogeneous retrieval evaluation
  • RAGAS for retrieval, faithfulness, and answer evaluation dimensions
  • Lost in the Middle for limits in how language models use long context
